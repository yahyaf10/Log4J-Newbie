Log4J-Newbie
============

*HelloWorld* app using Log4J for Newbie. 



![Log4J log](http://logging.apache.org/log4j/2.x/images/logo.jpg)

**Log4J v2.0 beta 3**

Installation
------------

- Make sure you added **Core** and **Api** .jars to your Java Project using your IDE.  


Requierements
-------------

- Java SE 7 and Log4J 2.0 beta 3
